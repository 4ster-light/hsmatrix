# hsmatrix
An alternative to the famous program written in C++ `cmatrix`.

## Usage
```bash
cabal run
```
> [!NOTE]
> For now this program is only tested on Linux.
> And it doesn't have the full functionality of `cmatrix` yet.

> [!IMPORTANT]
> You need to have installed:
> - `cabal`
> - `stack`
> - `ghc`
> In order to run this program.
